# Contributing to ArbRemote

We appreciate your interest in contributing to ArbRemote! 

## Scope
Contributions are welcome for:
- User Interface (UI) improvements and design polish.
- SEO optimizations and meta-tag refinements.
- Accessibility (A11y) enhancements.
- Bug fixes related to the Astro frontend or styling.

### Out of Scope
The following components are **proprietary** and are not handled in this repository:
- Data scraping and collection logic.
- Lead scoring and ranking algorithms.
- Backend automation pipelines.

## Getting Started
1. Fork the repo.
2. Create a feature branch.
3. Submit a Pull Request with a detailed description of your changes.

Thank you for helping us build the future of remote work arbitrage.
