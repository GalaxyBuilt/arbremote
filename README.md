# ArbRemote — Earn USD. Live Anywhere.

[![Astro](https://img.shields.io/badge/Astro-BC52EE?style=flat-square&logo=astro&logoColor=white)](https://astro.build/)
[![Cloudflare Pages](https://img.shields.io/badge/Cloudflare_Pages-F38020?style=flat-square&logo=cloudflare&logoColor=white)](https://pages.cloudflare.com/)

ArbRemote is the open-source landing page and distribution layer for the **Galaxy Arbitrage** remote job list. It is designed for high-intent conversion, highlighting the math behind geographic arbitrage for engineers and founders.

## About
ArbRemote empowers global talent to maximize their net savings by connecting them with USD-paying roles while living in high-quality, low-cost regions. 

> [!NOTE]
> The proprietary data pipeline, scraping engine, and lead scoring logic are closed-source and maintained in a separate private repository.

## Tech Stack
- **Framework**: Astro 4.0
- **Styling**: Vanilla CSS (CSS Variables for Theme Management)
- **Deployment**: Cloudflare Pages
- **Analytics**: Beehiiv + Pixel Tracking Placeholders
- **Typography**: Fraunces (Serif) & DM Mono (Monospace)

## Deploy
To deploy your own version:
1. Fork this repository.
2. Link your repository to Cloudflare Pages.
3. Use the following build settings:
   - **Framework Preset**: Astro
   - **Build Command**: `npm run build`
   - **Build Output Directory**: `dist`
   - **Node Version**: 20
